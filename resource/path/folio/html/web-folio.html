제작 시기에 맞춰 순서대로 나열해보았습니다.

개요

프로젝트 소개

NINIZ

반복되는 HTML 구조를 정적이 아닌 스크립트 동적으로 만들어 보고 싶었으며,
부끄럽게도 Javascript, Jquery 문법 차이도 모른 채 제작을 시도했던 사이트 입니다.
아이템 추가 수정을 관리자가 쉽게 접근할 수 있도록 엑셀로 구조화해보았습니다.

Slate(old)

웹 사이트를 제작하는 에디터를 만들어보고 싶었습니다.
도큐먼트를 조작하여 HTML,CSS를 만들어내거나,
내부 객체를 통해 구조를 메타화 하여 저장하거나 불러오기도 합니다.

이 프로젝트의 경험을 통해 JAVA 커리큘럼 과정의 Slate를 제작해보았습니다.
에디터 혹은 메타데이터를 통해 관리되기 보다는
HTML,CSS,JS 구조를 그대로 사용하여 도큐먼트를 조작 및 객체화, 랜더링하는 형태로 구성하였습니다.

Library

공공기관 도서관 사이트를 퍼블리싱 해보았습니다.

Responsive

반응협 웹 사이트를 퍼블리싱 해보았습니다.

Hyundai

Javascript, Jquery의 문법 차이를 알게 되어 Javascript로 제작하였습니다.
ES6(Class)의 객체지향 프로그래밍에 대해 고민하였고
슬라이더, 자동차 VR, 데이터 등을 객체로 접근하였습니다.
360도 촬영된 자동차 이미지를 Car, Color로 구분하여 접근합니다.
