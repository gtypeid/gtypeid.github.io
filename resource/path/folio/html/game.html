프로젝트 소개

Unity Project

커리큘럼 외 Unreal 엔진을 습득하며 배운 구조로 삼아 게임 콘텐츠를 제작해보았습니다.

Unreal Project

Unreal 엔진의 제공하는 클래스 구조를 이해하고, 적합한 컴포넌트에 기능을 구현하였습니다.
또한 프로퍼티를 외부로 노출하거나, 외부 엑셀 CSV파일 및
에디터 내의 아이템으로 콘텐츠를 만들어내고 리소스를 최대한 활용할 수 있는 구조를 고민해봤습니다.
스테이트 머신을 이용하여 상태를 전환 시켜 애니메이션, 전투, 시스템등 일관성 있는 작동을 구현해보았습니다.
